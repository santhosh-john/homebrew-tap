# Homebrew Tap
Homebrew tap for awsmgr and related tools.

## Installation

```bash
brew tap santhosh-john/tap
brew install awsmgr
```

## Usage
- awsmgr help
- awsmgr list
- awsmgr current
- awsmgr validate
- awsmgr doctor

For details, see the [main](https://github.com/santhosh-john/awsmgr) repository.
